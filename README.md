# Worlds 2018

Worlds 2018 score boards

## Getting started

Clone the repository:

```bash
git clone https://github.com/toinelin/worlds-2018.git
```

Duplicate .env.sample and rename it .env
Add you PANDASCORE token on .env

Install it and run:

```bash
cd worlds-2018
yarn
yarn start
```
